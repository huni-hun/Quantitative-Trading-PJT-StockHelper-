# 📈 StockHelper — 한국투자증권 자동 매매 봇

한국투자증권(KIS) OpenAPI 기반의 **국내·해외 주식 자동 매매 봇**.  
뉴스 감성 분석(LLM), 기술적 분석(멀티 지표), 모멘텀 전략, 트럼프 포스팅 실시간 모니터링을 **가중 점수 방식**으로 통합하여 매매 시그널을 생성한다.  
**웹 대시보드(Flask)** 에서 모든 설정을 GUI로 관리하고, 실시간 시그널·점수·로그·거래내역·보유종목을 확인할 수 있다.

## 빠른 시작

### 방법 A — `run.bat` 실행 (권장, Windows)

```
run.bat 실행 후 메뉴 선택:
  1. 웹 대시보드만 실행
  2. 매매봇만 직접 실행
  3. 대시보드 + 봇 동시 실행  ← 권장
```

3번 선택 시 대시보드가 먼저 뜨고 3초 후 봇이 자동 시작된다. 봇 상태가 대시보드에 실시간 반영된다.

```bash

# 터미널 2: 봇 (HTTP 콜백 모드)
set STOCKHELPER_SUBPROCESS=1
set DASHBOARD_URL=http://127.0.0.1:5000
python main.py
```

> 봇을 중지하려면 `Ctrl + C`

## 매매 전략 상세

봇은 **4개의 독립적인 전략**을 동시에 실행하고 가중 점수로 통합하여 최종 매매 결정을 내린다.

---

### 전략 1 — 뉴스 감성 분석 (`NewsSentimentStrategy`)

**목적:** 최신 뉴스에 반영된 시장 심리를 수치화하여 단기 방향성을 예측한다.

```
[뉴스 수집]
  국내 종목 → 네이버 금융 종목 전용 뉴스 페이지 크롤링
  해외 종목 → 네이버 뉴스 검색 (티커명 + 주가)
  최신 헤드라인 최대 MAX_HEADLINES건 수집
        ↓
[LLM 감성 점수 산출 -1.0 ~ +1.0]
  판단 우선순위:
  [HIGH] 실적 서프라이즈/쇼크, EPS Beat/Miss, 매출 성장/감소, 가이던스 상향/하향
  [HIGH] M&A, 대형 계약, 주요 파트너십, 임원 교체
  [MED]  업황 뉴스, 경쟁사 동향, 규제 변화, 소송/리스크
  [LOW]  일반 투자자 의견, 목표주가 변경, 단순 주가 언급

  반환: {"score": float, "confidence": float, "key_factors": [...], "reason": "..."}
        ↓
[신뢰도 보정]  adjusted = raw_score × max(0.5, confidence)
        ↓
[롤링 EMA 평균화]  ← 단일 뉴스 노이즈 제거
  rolling = α × adjusted + (1-α) × 이전_rolling  (α=0.4, 최근 5회 캐싱)
        ↓
[추세 방향성 보너스]  3회 연속 동일 방향 → 점수 × 1.1
        ↓
[시그널 생성]  rolling ≥ +임계값 → BUY / ≤ -임계값 → SELL
```

> **핵심:** EMA 평균화로 낚시성 헤드라인 노이즈를 제거하고, LLM 신뢰도가 낮으면 점수가 자동으로 0에 수렴한다.

---

### 전략 2 — 기술적 분석 (`DeadcatTechnicalStrategy`)

**목적:** 가격·거래량·추세 등 6개 지표의 점수를 합산하여 과매도/과매수 구간을 식별한다.

KIS API 일봉 OHLCV 1회 조회 후 아래 6개 지표를 병렬 계산한다.

| 지표 | 계산 방법 | BUY 기여 | SELL 기여 |
|---|---|---|---|
| **RSI** (14일, Wilder EWM) | RSI < 30 / RSI > 70 | +1.0 | -1.0 |
| **볼린저 밴드** (20일, ±2σ) | 하단 이탈 / 상단 돌파 | +1.0 | -1.0 |
| **MACD** (12/26/9) | 골든크로스 / 데드크로스 | +1.0 / +0.5 유지 | -1.0 / -0.5 유지 |
| **EMA 추세** (20/50/200) | 정배열(20>50>200) / 역배열 | +1.0 / +0.5 장기 위 | -1.0 / -0.5 장기 아래 |
| **거래량 급증** (20일 평균 1.5배↑) | RSI 방향 확증 시 | +0.5 | -0.5 |
| **Stochastic RSI** (14일) | K < 20 / K > 80 | +0.5 | -0.5 |

| 총점 | 시그널 |
|---|---|
| `≥ +2.0` | **BUY** |
| `≤ -2.0` | **SELL** |
| 그 외 | **HOLD** |

> **핵심:** EMA 200 하락 추세에서는 RSI 과매도가 와도 EMA 점수가 음수라 전체 합계가 낮아져 매수가 자연스럽게 억제된다.

---

### 전략 3 — 모멘텀 전략 (`MomentumStrategy`)

**목적:** 강한 상승 추세를 포착하여 탑승하고, 하락 추세에서 선제적으로 회피한다.  
전략 2(평균회귀)와 반대 성격의 **추세 추종** 전략으로, 서로 보완 관계다.

OHLCV는 전략 2와 공유 (추가 API 호출 없음).

| 지표 | 기여 |
|---|---|
| **EMA 20/50 크로스** | 골든크로스 +2.0 / 데드크로스 -2.0 / 정배열 유지 +1.0 / 역배열 -1.0 |
| **52일 신고가 근접** | 5% 이내 브레이크아웃 +1.5 / 90% 이상 +0.5 / -20% 이하 -1.0 |
| **N일 가격 모멘텀** (20일 수익률) | ≥+10% → +1.0 / ≥+3% → +0.5 / ≤-10% → -1.0 / ≤-3% → -0.5 |
| **ADX 추세 강도 필터** (14일) | ADX≥25: 점수×1.0 / 15~25: ×0.6 / <15(횡보): ×0.3 |

| ADX 보정 후 총점 | 시그널 |
|---|---|
| `≥ +2.5` | **BUY** |
| `≤ -2.5` | **SELL** |
| 그 외 | **HOLD** |

> **핵심:** ADX가 낮은 횡보장(방향성 없음)에서는 점수가 최대 70% 감쇠되어 가짜 추세 신호를 억제한다.

### 전략 4 — 트럼프 포스팅 모니터 (`TrumpMonitor`)

**목적:** 미국 시장에 즉각적인 영향을 미치는 트럼프 대통령 포스팅을 24시간 감시하여 거시적 위험을 반영한다.

```
[Nitter RSS (@realDonaldTrump) 폴링]
  TRUMP_POLL_INTERVAL초마다 (기본 30초), 24시간 상시 백그라운드 스레드
  봇 시작 시 기존 포스트 자동 스킵 (재시작 시 과거 글로 주문 발생 방지)
        ↓
[새 포스팅 감지 즉시 LLM 분석]
  시장 영향도 점수 [-1.0 ~ +1.0] + 키워드 + 이유 추출
        ↓
[TrumpSignalStore (싱글톤) 저장 → 전략 루프에서 매 사이클 참조]
```

| 점수 | 시그널 | 예시 키워드 |
|---|---|---|
| `≥ +0.35` | **BULLISH** | 감세, 규제완화, 기업 칭찬, 경제 호황 |
| `≤ -0.35` | **BEARISH** | 관세, 무역전쟁, 기업 비판, 금리 압박 |
| 그 외 / 신규 없음 | **NEUTRAL** | 일반 정치 발언 |

> Nitter 인스턴스 3개 자동 폴백.

## 매매 결정 로직

4개 전략의 시그널을 **가중 점수(Weighted Score)** 방식으로 통합한다.

### 가중치 구성

| 전략 | 가중치 | 역할 |
|---|---|---|
| 기술적 분석 (전략 2) | **40%** | 가격·거래량 기반 주요 시그널 |
| 뉴스 감성 분석 (전략 1) | **30%** | 펀더멘털·이벤트 방향성 |
| 모멘텀 전략 (전략 3) | **20%** | 추세 추종 확증 |
| 트럼프 시그널 (전략 4) | **10%** | 거시적 위험 필터 |

### 복합 점수 계산

```
composite = 기술(+1/-1/0) × 0.40
          + 감성(+1/-1/0) × 0.30
          + 모멘텀(+1/-1/0) × 0.20
          + 트럼프(+1/-1/0) × 0.10
```

| composite | 최종 결정 |
|---|---|
| `≥ +0.45` | **BUY** 시장가 매수 |
| `≤ -0.45` | **SELL** 시장가 매도 |
| `-0.45 ~ +0.45` | **HOLD** 관망 |

### 시나리오 예시

```
[강한 매수]  기술BUY(+0.40) + 감성BUY(+0.30) + 모멘텀BUY(+0.20) + 트럼프NEUTRAL(0)
             = +0.90 → BUY ✅

[엇갈린 신호] 기술BUY(+0.40) + 감성SELL(-0.30) + 모멘텀HOLD(0) + 트럼프NEUTRAL(0)
             = +0.10 → HOLD ✅  (확신 없음)

[트럼프 BEARISH 영향] 기술BUY(+0.40) + 감성BUY(+0.30) + 모멘텀HOLD(0) + 트럼프BEARISH(-0.10)
             = +0.60 → BUY ✅  (트럼프 단독 차단 불가, 점수만 감소)
```

> **기존 AND 룰과의 차이:** 전략이 완전히 동의하지 않아도 다수가 같은 방향을 가리키면 매매가 발생한다. 더 많은 기회를 포착하되, 단일 지표에 과의존하지 않는다.

## 리스크 관리

### 1단계 — `RiskManager` 자동 손절/익절

매 사이클에서 전략 실행 **이전**에 현재 포지션을 먼저 확인한다.

```
현재가 ≤ 진입가 - ATR×2.5  →  🛑 즉시 시장가 손절 매도
현재가 ≥ 진입가 + ATR×4.0  →  ✅ 즉시 시장가 익절 매도
(ATR 없으면: -7% 손절 / +15% 익절 고정 비율 사용)

일일 실현 손익 ≤ -DAILY_LOSS_LIMIT  →  당일 신규 매매 정지
```

매수 시 ATR 기반 포지션 사이징: `qty = (자본 × 2%) / (ATR × 2.5)`

### 2단계 — 웹 대시보드 수동 제어

| 기능 | 설명 |
|---|---|
| **⏹ 킬 스위치** | 전략 루프 즉시 차단. `▶ 봇 재개`로 재시작 없이 복구 |
| **🚨 패닉 셀** | 봇 정지 + 모든 보유 종목 즉시 시장가 청산 |
| **일일 손실 한도** | 오늘 손익률이 설정% 이하 → 봇 자동 정지 + 텔레그램 긴급 알림 |
